# RenduTDD

Pour installer le projet :

npm install

- Lancer les test : ``` npm run test ```
- Lancer la couverture de test : ``` npm run test:cov ```
- Lancer les test à chaque sauvegarde de l'IDE : ``` npm run test:watch ```
